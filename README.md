## 🚀 核心功能 (Features)

### 1. 🔪 表格智能拆分 (Smart Split)
* **痛点**：拆分表格需要开通会员，手动复制粘贴容易出错且耗时。
* **功能**：拖入一个总表（如《10月全员薪资表》），选择按“部门”或“姓名”列，**一秒钟拆分为 500 个独立文件**。
* **亮点**：保留原表格式，自动命名。

### 2. 🔗 多表极速合并 (Quick Merge)
* **痛点**：月底要将各部门发来的 50 个表格汇总成一个总表，复制粘贴到手酸。
* **功能**：选中所有分表拖入软件，**一键合并成一个总表**。
* **亮点**：自动去除重复表头，智能对齐数据。

### 对于开发者 (Developers)

如果你想自己打包或贡献代码，请按以下步骤操作：

#### 1. 环境准备
你需要安装 Python 3.10+ 和以下依赖库：
```bash
git clone [https://github.com/aaronzhuang19-sketch/Excel-Office-Killer.git]
cd Excel-Office-Killer
pip install -r requirements.txt
2. 运行源码
Bash

python main.py
3. 自行打包 (Build .exe)
如果你不信任发布的 exe，可以自己打包：

Bash

pip install pyinstaller
pyinstaller --noconsole --onefile --name="OfficeKiller" main.py

📝 常见问题 (Q&A)
Q: 为什么软件打不开？ A: 请确保你的系统是 Windows 10 或 Windows 11。如果双击没反应，请尝试右键 -> 以管理员身份运行。

Q: 支持 Mac 系统吗？ A: 目前仅打包了 Windows 版本。Mac 用户可以下载源码并通过终端运行 python main.py 使用。

Q: 处理几万行的大文件会卡死吗？ A: 软件采用多线程处理，大文件处理时界面可能会短暂显示“处理中”，请耐心等待，完成后会自动打开结果文件夹。
